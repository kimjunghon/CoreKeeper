# CoreKeeper

- **플레이 영상**
  
[https://youtu.be/liiP3Buqf9E](https://youtu.be/liiP3Buqf9E)

- 프로젝트 개요
    - C++과 Win32 API 기반으로 크래프팅 중심 게임 시스템 구현을 목표로 제작한 Core Keeper 모작 프로젝트입니다.
    - 타일 기반 맵 시스템과 크래프팅 중심의 아이템 제작 및 사용 시스템을  핵심으로 다양한 게임 플레이 기능을 구현했습니다.
    
- 개발 목표
    - 원작 게임 시스템을 분석하여 핵심 게임 플레이 구조 재현
    - 타일 배치 툴과 데이터 입출력 구조를 구현하여 맵 제작 및 관리 기능 구현
    - NPC 상호작용을 통한 퀘스트 콘텐츠 구현
